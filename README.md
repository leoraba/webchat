# Tools
React \

# Installation

create a `.env` file in the root folder with the following variables from Firebase\
REACT_APP_APIKEY=\
REACT_APP_AUTHDOMAIN=\
REACT_APP_PROJECTID=\
REACT_APP_STORAGEBUCKET=\
REACT_APP_MESSAGESENDERID=\
REACT_APP_APPID=


npm install

npm run start


# Build & Deploy

npm run build

firebase deploy
